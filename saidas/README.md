# Saídas

Pasta para documentos pontuais produzidos pelo MaxClaudePro.

## Subpastas criadas automaticamente

- `propostas/` — propostas comerciais em HTML
- `briefings/` — documentos de briefing
- `reunioes/` — atas de reunião
- `relatorios/` — relatórios de resultado e progresso
- `scripts/` — scripts de vendas
