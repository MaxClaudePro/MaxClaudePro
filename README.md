# MaxClaudePro

**O sistema operacional do seu negócio dentro do Claude Code.**

---

## O que é

MaxClaudePro é uma estrutura de arquivos que transforma o Claude Code no assistente central da sua operação. Ele aprende como o seu negócio funciona, como você escreve, quais são suas prioridades — e usa isso em tudo que produz.

Não é um software. É um sistema de contexto + comandos prontos que qualquer empresa pode instalar em minutos.

---

## Como instalar

1. Instale uma IDE com suporte ao Claude Code (VS Code, Cursor ou Antigravity IDE)
2. Instale o Claude Code na IDE: [claude.ai/code](https://claude.ai/code)
3. Cole o comando abaixo no chat do Claude Code:

```
Clona o https://github.com/MaxClaudePro/MaxClaudePro.git na pasta atual, entra nela e roda o instalar.
```

O setup dura cerca de 5-7 minutos e configura tudo automaticamente.

## Skills disponíveis

| Comando | O que faz |
|---|---|
| `/instalar` | Setup inicial — configura o sistema pro seu negócio |
| `/salvar` | Salva uma informação na memória do sistema |
| `/atualizar` | Atualiza os arquivos de memória com mudanças recentes |
| `/status` | Mostra tudo que está configurado |
| `/carrossel` | Cria carrossel para Instagram com visual da marca |
| `/legenda` | Gera legenda pronta para posts |
| `/email-marketing` | Escreve emails de marketing, prospecção e comunicados |
| `/seo` | Palavras-chave, meta tags e artigos otimizados |
| `/blog` | Artigo completo para blog |
| `/proposta` | Proposta comercial em HTML pronta pra enviar |
| `/followup` | Mensagem de follow-up para leads e clientes |
| `/script-vendas` | Script de abordagem, qualificação e fechamento |
| `/responder` | Responde mensagens de clientes e leads |
| `/avaliacoes` | Responde avaliações no Google, Instagram e outros |
| `/anuncio-meta` | Textos para Facebook e Instagram Ads |
| `/anuncio-google` | Headlines e descrições para Google Ads |
| `/briefing` | Levantamento completo de requisitos de projeto |
| `/relatorio` | Relatório de resultados ou progresso |
| `/reuniao` | Ata de reunião com decisões e ações |

## Estrutura de pastas

```
/
├── CLAUDE.md              ← Cérebro do sistema (não editar manualmente)
├── _memoria/
│   ├── empresa.md         ← Quem é a empresa
│   ├── preferencias.md    ← Tom de voz e comunicação
│   └── estrategia.md      ← Prioridades e foco atual
├── identidade/
│   └── design-guide.md    ← Cores, fontes, logo
├── marketing/             ← Conteúdo produzido (carrosséis, posts, artigos)
├── saidas/                ← Documentos (propostas, atas, relatórios)
├── dados/                 ← Arquivos para analisar
├── scripts/               ← Automações
└── .claude/skills/        ← Skills do sistema
```
